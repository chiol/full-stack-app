<h1 align="center">full stack monorepo</h1>

# Packages

- mobile - react native
- web - react + nextjs
- backend - hasura + strapi + nestjs

# Frontend

![screenshot](./imgs/screenshot.png)

## install

```bash
yarn install --check-file
```

## Usage

```bash
# andriod
yarn android
# ios
yarn ios
# web
yarn web
```
